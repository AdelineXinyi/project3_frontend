# Project 3 Frontend

Vue.js frontend for the multi-agent scientific discovery system.

The frontend visualizes:
- real-time agent transitions
- streaming Server-Sent Events (SSE)
- generated scientific report previews
- generated matplotlib figures
- downloadable DOCX reports

---

## Features

- Real-time agent status updates
- Streaming backend communication
- GRPO-style optimization toggle
- Multi-round optimization configuration
- Figure preview rendering
- Report preview panel

---

## Setup

```bash
conda activate scisci

python -m http.server 5500
```
